A modern and responsive Weather Application built using HTML, CSS, and JavaScript that provides real-time weather updates for any city using the OpenWeather API.

Users can search any city and instantly view:

🌡️ Current temperature
☁️ Weather condition (Rain, Sunny, Cloudy, etc.)
💧 Humidity level
💨 Wind speed
📅 5-day weather forecast

The app also includes dynamic visual effects such as:

🌧️ Rain animation during rainy weather
☀️ Sun animation during clear weather
☁️ Moving cloud animation during cloudy weather
🎨 Background changes based on weather conditions

Additional features:

🔍 Search by city name
⏳ Loading spinner while fetching data
📱 Mobile-friendly responsive design
✨ Clean modern UI with smooth animations

This project demonstrates skills in:

API integration
JavaScript DOM manipulation
Asynchronous programming (fetch)
Responsive web design
UI/UX enhancements with animations
